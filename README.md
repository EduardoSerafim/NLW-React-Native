# Aplicativo de cardapio em React Native

Aplicativo desenvolvido durante a trilha de React Native da Next Level Week da rocketseat, consiste em um aplicativo de cardapio onde o usuário pode adicionar produtos no carrinho e realizar o pedido através do whatsapp do estabelecimento

Para o desenvolvimento do aplicativo foi usado React Native com TypeScript, Tailwind para a estilização, além de bibliotecas como CLSX, keyboard-aware-scroll-view, Async Storage e Zustand
